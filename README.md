## Local Deploying

1. Clone the repo
   ```
   git clone https://github.com/DARKEMPIRESL/YT-Channel-DL
   ```

2. Enter the directory
   ```
   cd YT-Channel-DL
   ```
  
3. Install all requirements using pip.
   ```
   pip3 install -r requirements.txt
   ```

4. Run the file
   ```
   python3 bot.py
   ```
